# Página Web: Ejercición de cromatización

Cromatización de una misma página según la teoría y psicología del color:

- **Colores azules:** Tranquilidad.
- **Colores negros y morados:** Tristeza.
- **Colores rojos:** Miedo.
- **Colores Verdes:** Armonía.
